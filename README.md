# JWST Explorations

A series of Python notebooks exploring publicly available data from the James Webb Space Telescope via STScI's MAST archive.

## Notebooks

**01 - Southern Ring Nebula (NGC 3132)**  
Fetches a calibrated NIRCam F356W mosaic from MAST and visualizes the data using ZScale interval stretching — the standard astronomical display algorithm. Demonstrates the difference between raw pixel values and a scientifically meaningful image display.

Data source: MAST Archive · STScI · NASA  
Libraries: astropy · numpy · matplotlib
